# Clipboard Manager

Clibor 風のクリップボード履歴マネージャー（Windows 向け Electron アプリ）

## 機能

| 機能 | 説明 |
|------|------|
| **クリップボード履歴** | コピーしたテキストを自動記録（最大 10,000 件） |
| **定型文** | よく使う文章をグループ別に登録・ワンクリック貼り付け |
| **FIFO / LIFO** | 連続コピー → 連続貼り付けモード |
| **テキスト整形** | 検索置換・正規表現による整形 |
| **検索** | 履歴・定型文のリアルタイム検索 |
| **ピン留め** | 重要な履歴を削除から保護 |
| **ホットキー** | `Alt+V`（変更可）でポップアップ表示 |
| **テーマ** | 4 種類の配色プリセット |
| **常駐** | システムトレイに常駐、ローカル保存のみ |

## 必要環境

- Windows 10 / 11
- Node.js 18 以上

## 起動方法

```bat
start.bat
```

または:

```bash
npm install
npm start
```

### Electron のインストールエラーが出る場合

`Electron failed to install correctly` と表示されたら、次を実行してください:

```bash
npm run fix-electron
npm start
```

`npm install` 時に `postinstall` で自動修復を試みますが、ネットワーク環境によっては上記が必要な場合があります。

## 基本操作

1. 起動後、タスクトレイにアイコンが表示されます
2. テキストをコピーすると自動で履歴に追加されます
3. **Alt+V** で履歴ウィンドウを表示/非表示
4. 履歴をクリック → クリップボードにセットして自動貼り付け（Ctrl+V 送信）
5. トレイアイコンをダブルクリックでも表示できます

### キーボード操作

- `↑` / `↓` — 履歴・定型文の選択
- `Enter` — 選択項目を貼り付け
- `Esc` — ウィンドウを閉じる

## 設定

「設定」タブから以下を変更できます:

- 表示ホットキー（例: `Alt+V`, `Ctrl+Shift+V`）
- 履歴の最大件数（100〜10,000）
- 貼り付けモード（通常 / FIFO）
- テーマ（デフォルト / ダーク / グリーン / パープル）
- カーソル位置に表示
- 連続重複の無視
- Windows 起動時の自動起動

## データ保存

すべてのデータはローカル PC 内に保存されます（`electron-store`）。外部サーバーへの送信はありません。
