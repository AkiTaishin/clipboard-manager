# Clipboard Manager

Clibor を参考にした、Windows 向けクリップボード履歴ツールです。  
コピーしたものを覚えておいて、あとからサッと呼び出す。それだけの話。

タスクバーには出ず、通知領域（タスクトレイ）にひっそり常駐します。CMD を閉じたらアプリも終わる、みたいなことは起きないようにしてあるので、普段使いは `start.bat` 一発で OK。

---

## これ何ができるの？

だいたいこんな感じ。

- **クリップボード履歴** — テキストも画像も自動で記録（最大 10,000 件まで設定可）
- **定型文** — よく使う文をグループ分けして登録、クリックで貼り付け
- **FIFO モード** — 連続コピー → 連続貼り付け（地味に便利）
- **テキスト整形** — 検索置換・正規表現
- **重複まとめ** — 同じものを何度コピーしても `×3` みたいに回数表示
- **URL / メール / パス自動認識** — バッジ表示＆ワンクリックで開く
- **コピーのみ / 貼り付け切り替え** — クリックしたときの動作を選べる
- **プレーンテキスト貼り付け** — 書式なしでサッと貼る
- **数字キー即貼り付け** — ウィンドウ表示中に `1`〜`9` で上から貼れる
- **ピン留め・検索・テーマ** — 地味にそろってる

データは全部ローカル PC 内。外部に送ったりはしません。

---

## 動かすのに必要なもの

- Windows 10 / 11
- Node.js 18 以上（初回セットアップ時）

---

## 起動方法

### 普段使い（これを使って）

```bat
start.bat
```

または `start.vbs` をダブルクリック。

- CMD は一瞬出てすぐ消える（または出ない）
- **タスクトレイにアイコンが残る**
- タスクバーには出ない
- CMD を×で閉じてもアプリは生きてる

### 開発・デバッグ用

```bash
npm install
npm start
```

こっちはコンソールが残る。開発するときだけでいい。

### Electron が怒られたとき

`Electron failed to install correctly` と出たら:

```bash
npm run fix-electron
```

それでもダメなら `start.bat` をもう一度。初回セットアップも兼ねてる。

## 使い方（ざっくり）

1. 起動する → タスクトレイに脳みそアイコンが出る
2. いつも通りコピーする → 勝手に履歴に入る
3. **Alt+V** または **Ctrl を素早く2回** で履歴ウィンドウを開く
4. 項目をクリック → 貼り付け（設定で「コピーのみ」にもできる）
5. 終わったら **Esc** で閉じるか、もう一度 Alt+V

ウィンドウを開くたび、**履歴タブが最初に出る**ようになってます。

### キーボードショートカット

| キー | 動作 |
|------|------|
| `Alt+V` | 表示 / 非表示（変更可） |
| `Ctrl` × 2（素早く） | 表示 / 非表示 |
| `1`〜`9` | 上から N 番目を即貼り付け |
| `↑` `↓` | 選択移動 |
| `Enter` | 選択項目を実行 |
| `Esc` | ウィンドウを閉じる |

### 履歴の操作

| 操作 | 説明 |
|------|------|
| クリック | 貼り付け or コピーのみ（ツールバーで切替） |
| 📋 | コピーのみ |
| 📝 | プレーンテキストで貼り付け |
| ↗ | URL・メール・パスを開く |
| 📌 | ピン留め（削除から守る） |
| ⭐ | 定型文に登録 |

## 設定の初期値

いじる前はだいたいこうなってます（`src/defaults.js` が正）。

| 項目 | デフォルト |
|------|-----------|
| ホットキー | `Alt+V` |
| ダブル Ctrl | ON（400ms） |
| クリック時 | 貼り付け |
| プレーンテキスト貼り付け | ON |
| 重複まとめ | ON |
| 画像クリップボード保存 | ON |
| 履歴の最大件数 | 1,000 件 |
| カーソル位置に表示 | ON |
| 起動時に最小化 | ON |
| Windows 起動時に自動起動 | OFF |
| テーマ | 青（デフォルト） |

設定タブからいくらでも変えられます。Windows 起動時の自動起動を ON にすると `start.vbs` 経由でログイン時に立ち上がります。

---

## 終了方法

タスクトレイのアイコンを右クリック → **「終了」**

ここだけはちゃんと終わらせてください。

## ファイル構成（気になった人向け）

```

├── start.bat          # 普段使いの起動（おすすめ）
├── start.vbs          # コンソール非表示起動
├── start-hidden.ps1   # VBS がダメなときの予備
├── src/
│   ├── main.js        # メインプロセス
│   ├── store.js       # 履歴・設定の保存
│   ├── defaults.js    # デフォルト設定
│   ├── double-ctrl.js # Ctrl×2 検出
│   ├── content-detect.js
│   └── assets/icon.png
└── scripts/
    ├── install-electron.js
    └── launch-background.js
```
